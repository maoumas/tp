# Generador de Audio con Azure TTS y GitHub OAuth

## Instalación
1. Clona el repositorio:
   git clone https://github.com/tuusuario/tts-web-module.git
2. Instala dependencias:
   npm install
3. Configura .env con tus credenciales.
4. Ejecuta:
   npm start

## Despliegue en Azure App Service
- Sube el proyecto a GitHub.
- Conecta Azure App Service con tu repo.
- Configura variables de entorno en Azure.
- Activa HTTPS (Azure lo hace automáticamente).
