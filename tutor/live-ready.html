The Master Studio Guide

@TranslatedPressDE Workflow

Phase 1: Research

Finding the right story.

01
Select Topic

Find a current news item interesting to older, retired Germans.

  • Critical of the Left
  • Supportive of Donald Trump
02
Gather Sources

Collect credible links, primarily from YouTube.

Phase 2: Scripting

Using NotebookLM.

03
Feed the AI

Create a new Notebook in NotebookLM and upload all your research sources.

04
Generate Script & Mark It

Ask for a teleprompter script. You must add these marks:

# Headline Title
* Narrator Note (Not spoken)
Plain text for speaking.

Save this as a "Clean Text File".

05
Get Visual Data

Ask NotebookLM for:

  • Slide Content (Titles, Prompts)
  • YouTube Source List (with timestamps)

Phase 3: Technical Setup

Preparing the Studio.

Teleprompter
  1. Go to zmau.dev/admin
  2. Paste your "Clean Text File".
  3. Generate file.
  4. Upload to GitHub subdirectory.
Visuals (Gemini)
  1. Open Gemini (Canva Mode).
  2. Input Slide Content from Phase 2.
  3. Create & Download Google Slides.
  4. Copy the Slides Link.
Control Center
  1. Go to zmau.dev/gen-kz
  2. Fill in required fields.
  3. Generate & Save the file locally.

Process Complete

You are ready to record or go live!
